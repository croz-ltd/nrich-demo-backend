-- ENCRYPT

Currently, annotations are not working in native mode because of: https://github.com/spring-projects-experimental/spring-native/issues/1712

When using decrypt use the value returned from encrypt endpoint.

curl -v --location --request GET http://localhost:8080/encrypt/encrypt-result\?value\=result
curl -v --location --request GET http://localhost:8080/encrypt/decrypt-argument\?value\=l5OvNFOj1KO3xIXmaeK9rjOLKWYGM0z9P9yVLAiI8rk=
curl -v --location --request POST http://localhost:8080/encrypt/encrypt-nested-result -H 'Content-Type: application/json' -d '{"nestedValue": "value" }'
curl -v --location --request POST http://localhost:8080/encrypt/decrypt-nested-argument -H 'Content-Type: application/json' -d '{"nestedValue": "9opiQN_5iiwMxoXkzYn4CYdW5QNg_aeOtlhOpwuiY0I=" }'
curl -v --location --request GET http://localhost:8080/encrypt/encrypt-result-defined-in-properties\?value\=result
curl -v --location --request GET http://localhost:8080/encrypt/decrypt-argument-defined-in-properties\?value\=r0zGxBATkA0LKx1iaYEMzNQSpVWvEs3mEG1L7Q-no78=
